API Introduction

The SolScanner Enterprise API provides programmatic access to our proprietary risk engine. Direct-to-ledger indexing ensures sub-50ms latency for critical authority checks.

01. Authentication

Include your API key in the header of all requests.

BASH
curl -X GET https://api.solscanner.io/v1/token/audit/So11111111111111111111111111111111111111112 \
  -H "X-API-KEY: sk_live_51M..." \
  -H "Content-Type: application/json"

02. Token Audit Response

The audit endpoint returns a recursive analysis of the token's metadata and authority tree.

JSON
{
  "mint": "So111...",
  "risk_score": 95,
  "authorities": {
    "mint": null, // Safe
    "freeze": null, // Safe
    "metadata": "revoked"
  },
  "liquidity": {
    "locked": true,
    "dex": "Raydium",
    "amount_usd": 450291.22
  }
}
